node >= 12 is required

## Start client

`$ cd client`

#### Install dependencies

`$ npm install`

#### Run dev environment

`$ npm start`

## Start server

`$ cd server`

#### Install dependencies

`$ npm install`

#### Run dev environment

`$ npm start`

## Start client and server with docker and docker-compose

`docker` and `docker-compose` is required

`$ docker-compose up --build`
